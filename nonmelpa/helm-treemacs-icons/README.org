** Configuration
*** Melpa (TBD)
*** Manual
    Make sure that you have both =helm= and =treemacs= installed clone the repo
    and put it on your path. Make sure that the enable call is performed before
    loading =helm=.

    #+BEGIN_SRC emacs-lisp
    (helm-treemacs-icons-enable)
    #+END_SRC
** Supported commands:
    - =helm-recentf=
    - =helm-mini=
    - =helm-find-file=
    - =helm-buffers-list=
    - =helm-locate=
    - =helm-fasd=
    - =helm-projectile-*=
